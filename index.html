!
Configuración requerida
Para conectar las estadísticas necesitás configurar 3 variables de entorno en Netlify:
META_ACCESS_TOKEN — Token de acceso de Meta (desde Graph API Explorer)
META_FB_PAGE_ID — ID de la página de Facebook
META_IG_USER_ID — ID de la cuenta de Instagram profesional

Permisos necesarios: pages_show_list, pages_read_engagement, instagram_basic, instagram_manage_insights, read_insights
